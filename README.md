# Sales Forecasting Application (sales-fc)

## Project Overview This project is an end-to-end machine learning solution designed to predict future sales performance. It addresses the business problem of inventory and resource planning by providing accurate forecasts based on historical data.

## Key Features

Time-Series Modeling: Utilizes Facebook Prophet to handle seasonality and holiday effects in sales data.

Interactive Dashboard: Deploys the model using Streamlit, allowing users to adjust forecast horizons and visualize trends dynamically.

Data Processing: Includes a pipeline for cleaning and structuring raw retail transaction data.

## Tech Stack

Language: Python

Libraries: Pandas, Prophet, Streamlit, Plotly

Data Source: Store Sales Time Series (Kaggle)
